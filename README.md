# 🍔 Food Delivery Website

A modern and responsive food ordering platform built using **HTML, CSS, and JavaScript** that allows users to explore menus, discover popular dishes, and place food orders through an intuitive and user-friendly interface.

## ✨ Features

* 🍽️ Browse and explore a variety of food items
* 🔍 Search and filter dishes by category
* 🛒 Add and remove items from the cart
* 💰 View order summary and total price
* 📱 Fully responsive design for all devices
* ⚡ Smooth animations and interactive UI
* 🎨 Modern and user-friendly interface

### 1. Clone the repository

```bash

```

### 2. Navigate to the project directory

```bash
cd food-delivery-website
```

### 3. Open the project

Simply open the `index.html` file in your preferred browser.
